## ðŸ“Š SEO Strategy for Course Pages

### Key Principles
1. **Intent**: Users searching for "{topic} course" want to learn, enroll, and verify quality
2. **Keywords**: Include course name, topic, level, and outcome-focused terms
3. **Benefits**: Lead with "Learn," "Master," "Build," "Get certified"
4. **Social Proof**: Mention ratings, students, and outcomes
5. **CTR Optimization**: Clear value proposition in title and description

### URL Slug Pattern
```
/courses/{course-name-lowercase-with-hyphens}
/courses/react-complete-guide
/courses/python-data-science-bootcamp
/courses/web-design-from-scratch
```

### Meta Title Formula (50-60 characters)
```
Pattern: {Course Name} | Learn {Topic} [+ Outcome/Level]
Examples:
- React Complete Guide | Learn Web Development
- Python for Data Science | Master Advanced Analytics
- UI/UX Design Masterclass | Professional Certificate
```

### Meta Description Formula (140-160 characters)
```
Pattern: Learn {topic}. Master {key skills}. This course includes {features}. Enroll now for {benefits}.
Examples:
- Learn React from scratch. Master components, hooks, and state management. 50+ projects, lifetime access.
- Master Python for data analysis. Learn pandas, NumPy, matplotlib. 30+ real-world projects. Certificate.
```

### Primary Keyword Format
```
{Course Topic} + Intent Modifiers
- course online
- learn from scratch
- for beginners
- master in {duration}
- certification
```

## How to Generate SEO Elements Dynamically

### Option 1: Database-Driven (Recommended)
Store course metadata in the database with SEO fields:

```javascript
// Backend - Course Model Enhancement
const courseSchema = {
  // ... existing fields
  seoSlug: String,               // Unique URL slug
  seoTitle: String,              // 50-60 chars
  seoDescription: String,        // 140-160 chars
  seoKeywords: [String],         // Array of keywords
  seoPrimaryKeyword: String,     // Main keyword
  h1Heading: String,             // H1 for page
  courseOutcomes: [String],      // Learning outcomes
  targetAudience: String,        // Who should take this
};

// SEO Migration Query
db.courses.updateMany({}, {
  $set: {
    seoSlug: function(title) {
      return title.toLowerCase()
        .replace(/[^a-z0-9]+/g, '-')
        .replace(/(^-|-$)/g, '');
    }
  }
});
```

### Option 2: Auto-Generate in Frontend
```typescript
// frontend/lib/seo/courseMetadata.ts
import { generateCourseMetadata } from '@/lib/seo/metadata';

export async function generateCoursePageMetadata(courseId: string) {
  const course = await fetchCourse(courseId);
  
  const slug = course.title
    .toLowerCase()
    .replace(/[^a-z0-9]+/g, '-')
    .replace(/(^-|-$)/g, '');
  
  const keywords = generateKeywords(course);
  
  return generateCourseMetadata({
    title: course.title,
    description: course.description,
    keywords,
    slug,
    category: course.category,
    level: course.level,
    price: course.price,
    rating: course.rating,
  });
}
```

### Option 3: Use This Reference Document
Manually update each course's layout.tsx with elements from this document:

```typescript
// frontend/app/courses/[id]/layout.tsx
import { COURSE_SEO_ELEMENTS } from '@/lib/seo/courseElements';
```
